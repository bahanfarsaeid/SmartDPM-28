# SmartDPM‑28  
### A Cleaned, Standardized, and Downsampled Pressure‑Map Dataset for Sleep Posture Classification

SmartDPM‑28 is a fully cleaned, standardized, and downsampled pressure‑map dataset derived from the MIT‑BIH Sleep Posture Database.  
It contains **20,024 samples**, each represented as a **28×28 pressure map**, optimized for lightweight CNN models, embedded systems, and real‑time sleep posture analysis.

This dataset is developed by the **SmartDPM Team** as part of a broader project on intelligent medical mattresses and pressure‑mapping technologies.

---

## 🚀 Key Features

- **20,024 high‑quality samples**
- Cleaned and standardized from raw MIT‑BIH pressure data
- All frames resized from **32×64 → 28×28**
- Corrupted or incomplete frames automatically repaired
- Compressed NPZ format for fast loading and small package size
- Ideal for:
  - CNN training
  - Edge AI (Raspberry Pi, Jetson Nano, ESP32‑S3)
  - Sleep posture classification
  - Medical device prototyping
  - Research and academic projects
